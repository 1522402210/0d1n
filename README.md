## to run:

require libcurl-dev or libcurl-devel(on rpm linux based)

$ git clone https://github.com/CoolerVoid/0d1n/

 
need libcurl to run

  
$ sudo apt-get install libcurl-dev

if rpm distro

$ sudo yum install libcurl-devel

$ make
